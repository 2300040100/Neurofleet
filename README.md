# NeuroFleet AI Powered Urban Fleet and Traffic Intelligence

**NeuroFleet** is a smart fleet management system designed to help organizations efficiently monitor, manage, and optimize their vehicle fleets using modern web technologies.

The application provides a centralized platform to track vehicles, manage fleet data, and improve operational efficiency.

---

## Features

*  Fleet overview dashboard
*  Vehicle management (add, update, remove vehicles)
*  Real-time or simulated tracking support
*  Organized data management
*  Responsive and user-friendly UI
*  Fast performance with modern frontend tools
